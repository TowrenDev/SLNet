Quick Start Guide

                        

                            

                                
1

                                

                                    
Installation

                                    
Set the dedicated server directory in the Settings tab to point to your SCP: Secret Laboratory server installation.

                                

                            

                            

                                
2

                                

                                    
Configuration

                                    
Configure your server settings in the Gameplay Config tab and set up admin permissions in the Admin Config tab.

                                

                            

                            

                                
3

                                

                                    
Start Server

                                    
Use the Start button in the header to launch your server. Monitor the console for startup messages.

                                

                            

                            

                                
4

                                

                                    
Monitoring

                                    
Use the Status tab to monitor server performance and player activity in real-time.

                                

                            

                        

                    

                    

                        
Frequently Asked Questions

                        

                            

                                
How do I update my server?

                                
Updates are distributed by steam automatically, to participate in any betas go to the SCP:SL Dedicated Server in your steam library, click the little cog Settings -> Properties -> Betas.

                            

                            

                                
How do I add plugins?

                                
Plugin support is coming in a future update. Check the Plugins tab for more information.

                            

                            

                                
How do I add admins to my server?

                                
Go to the Admin Config tab and add Steam IDs to the Admin Steam IDs list.

                            

                            

                                
How do I change the server port?

                                
The port 7777 is universal and currently can not be changed for simplicity.
